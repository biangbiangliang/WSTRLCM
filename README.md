# WSTRLCM
The code of "Ring Local Contrast Measure Weighted by Spatial–Temporal Prior for Infrared Small Target Detection".

The details of our paper is:

L. Hu, D. Yang, D. Zhao, D. Tian, J. Chen and X. Wang, "Ring Local Contrast Measure Weighted by Spatial–Temporal Prior for Infrared Small Target Detection," in IEEE Geoscience and Remote Sensing Letters, vol. 23, pp. 1-5, 2026, Art no. 6005605.
